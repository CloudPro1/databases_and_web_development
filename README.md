# databases_and_web_development
Для информационной системы “Троллейбусный парк” разработана бизнес-модель, с ее внешними и внутренними пользователями, UML-диаграмма, карточки вариантов использования, файловая система, системная архитектура информационной системы, логическая модель базы данных. 

Используемые языки: Python, MySQL. Языки разметки: html, css.

------
Основные файлы для запуска находятся в папке main_curs.
В папке trolleybus_depot находятся все нужные таблицы для БД. В файле driver_salary_report.sql находится код процедуры, требуемый в проекте. Логическая модель БД находится на стр. 20 в документе "Козачук П.П. Троллейбусный парк.pdf".

Разделы дальше авторизации не будут работать, если вы не подключите свою БД в папке data. Поэтому в файле "Интерфейс проекта" можно посмотреть как все выглядит и работает, не запуская код.
