# databases_and_web_development
Курсовая работа.
Для информационной системы “Троллейбусный парк” разработана бизнес-модель, с ее внешними и внутренними пользователями, UML-диаграмма, карточки вариантов использования, файловая система, системная архитектура информационной системы, логическая модель базы данных. 

Используемые языки: Python, MySQL. Языки разметки: html, css.
